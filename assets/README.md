# Assets Directory

Place your banner images, GIFs, or other visual assets here.

## Recommended Assets:

1. **banner.gif** or **banner.png** - Header banner for your profile (recommended size: 1200x300px)
2. **profile-*.png** - Additional profile images if needed

## Usage in README:

Reference assets like this:
```markdown
<img src="https://github.com/sridharankaliyamoorthy/sridharankaliyamoorthy/blob/main/assets/banner.gif?raw=true" alt="Banner" />
```

The `?raw=true` parameter ensures GitHub serves the raw file instead of the HTML page.
